再拾Unity

发表于 2014-05-18 分类于 Interest

周末在家研究了下Unity。

目前做独立游戏，最合适的领域莫过于手游。之前有关注过Unity，但注意力很快就转移到了Cocos上。倒非是因为后者源码基于C++，而是天然地对开源的东西更有好感，可控可扩展也让人舒心。加之之前一直认为Unity略显复杂，收费更是产生隔阂。

对Cocos了解多了，一些显而易见的缺点也就自然无法忽视。完美主义倾向让我希望找一个替代品，Unity重新映入眼帘。

Unity目前有免费版，而且可以免费发布。官方资料很丰富，网上的教程也很多，初学者不会有障碍。

两天体验下来，整体感觉很好。就像大家对比Cocos2d-x和Unity时所总结的，前者是游戏引擎，后者是游戏开发解决方案。Unity编辑器功能丰富而强大，脚本可以使用C#和JavaScript，但推荐使用C#，配合MonoDeveloper，开发起来得心应手。

接下来的日子，会同时关注这两款引擎。但在开发时，会优先考虑Unity。同样有兴趣做独立游戏的同学，不妨也关注一下。
